# Hotel Management System

## Overview

Hotel Management System is a desktop application developed using Python, Tkinter, and MySQL. It helps manage hotel operations such as customer records, room bookings, room details, and user authentication through an easy-to-use graphical interface.

## Features

* User Login and Registration
* Customer Management
* Room Booking Management
* Room Details Management
* Hotel Record Management
* MySQL Database Integration

## Technologies Used

* Python
* Tkinter
* MySQL

## Project Structure

* customer.py
* details.py
* hotel.py
* login.py
* register.py
* room.py

## How to Run

1. Install Python.
2. Install MySQL and create the required database.
3. Update database credentials in the project files.
4. Run `hotel.py`.
